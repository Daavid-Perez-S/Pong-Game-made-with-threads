
JUEGO PONG
*************************

---------------
Instrucciones -
---------------


1.- Antes que nada, tiene que tener instalado en su PC el JRE de Java Oracle para poder
	ejecutar el .JAR del juego, sino lo tiene no podrá ejecutar el juego. Si lo tiene puede continuar al paso 2.
2.- Instale las fuentes de la carpeta "Fonts" que se encuentra en la raiz del proyecto, es decir, al inicio.
3.- No borres nada de la carpeta "PongFiles".
4.- Cuando inicie el juego, inserte 2 nombres para los distintos jugadores.
5.- Seleccione un nivel de dificultad.
6.- Empieza a jugar :D


----------------------------------------------------------------------------------------------------------------

PONG GAME
*************************

--------------
Instructions -
--------------

1.- First, you must have installed on your PC the Oracle's Java JRE to can run the game .JAR, if you
	don't have it you can't run this game. If all is alright you can go to the step 2.
2.- Install the fonts localized in the folder called "Fonts", it's in the beginning of the project.
3.- Not delete any files of the folder "PongFiles".
4.- When you start the game, insert 2 names for the differents players.
5.- Choose a difficulty level.
6.- Let's go to play  :D
